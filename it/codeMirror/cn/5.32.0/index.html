CodeMirror是运行在浏览器上的一个基于JavaScript实现的通用、多功能编辑器工具。 而且特别适合编写代码，能够满足多种编程语言的编写， 并且能通过增强组件来实现一些高级编辑功能。

现在CodeMirror支持丰富的可编程的API及根据css来编写多种主题， 因此你可以更加自由的自定义配置CodeMirror来兼容你的应用，你也可以通过新的函数功能来扩展你的应用。

特性

  • 创造性的支持超过一百种语言
  • 一个强大的, 可组合的 语言模式 系统
  • 自动完成 (XML)
  • 代码折叠
  • 可配置 快捷键绑定
  • Vim, Emacs, 和 Sublime Text 快捷键绑定
  • 支持查找和替换
  • 支持大括号 和 标签成对匹配
  • 支持分割视图显示
  • 语法错误标记及提示
  • 混合的字体大小和样式
  • 多种主题
  • 支持窗口大小改变时内容自适应
  • 行内 和 区块 小部件支持
  • 断点样式
  • 给选中范围的文字设置样式，只读, 或者不可分割
  • Bi-directional 文本支持
  • 更多其他方法 和 增强组件...

社区

CodeMirror 是一个开源软件，在MIT 协议下分享和使用。 它是一个编辑器工具，已被Chrome和Firefox浏览器， Light Table， Adobe Brackets, Bitbucket，和 其他很多项目作为开发者工具使用。

开发进程和bug追踪在github (一个兼容git仓库管理的工具)。 请先 阅读这些说明 后再提交bug。使用 pull requests来提交修复。 所用贡献的代码使用的协议必须和CodeMirror使用的MIT协议一致。

围绕这个项目的讨论社区已经建成在论坛中。 并且也有一个codemirror的声明列表， 这是唯一的最重要的公告(新版本亦适用). 如果有需要，你可以直接联系开发者。 我们致力于成为一个宽容的受欢迎的开发者社区。为了说明这一点，我们制定了一套适用于项目沟通的行为准则。

在我们的wiki上维护一个不属于主分发的CodeMirror相关软件列表。 你可以随时添加您的项目。

浏览器支持

在标准模式下，支持以下 桌面 版本的浏览器, (推荐使用HTML5 <!doctype html>):

Firefox4及以上
Chrome任何版本
Safari5.2版本及以上
Internet Explorer8及以上
Opera9及以上

对现代移动浏览器的支持是实验性的。在最新版本的iOS上的浏览器及Android上的Chrome浏览器上运行效果更好。
